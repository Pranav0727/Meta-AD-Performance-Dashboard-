This dashboard analyzes the performance of Meta ad campaigns across key KPIs, including impressions, clicks, engagements, conversions, and ad spend. 
It provides an end-to-end funnel view—from awareness and engagement to conversions—along with demographic, geographic, and time-based insights to support data-driven marketing decisions.
